### Explore US Bikeshare Data for the Past Year! 

In the <Link href="#shellHeader">terminal</Link> below, you can run SQL queries on rideshare data for the past year. All queries are run through <Link target="_blank" href="https://duckdb.org/2021/10/29/duckdb-wasm">DuckDB-Wasm</Link>, an in browser database that you can query with SQL.


For a list of possible cities, type:

`.files LIST` 

And then query the file directly:

```
SELECT * FROM 'boston.parquet' LIMIT 100
```

Enjoy! If you'd like to download all data for a city, you can download the parquet files below:
- [Boston](/boston_recent_year.parquet)
- [Chicago](/chicago_recent_year.parquet)
- [DC](/dc_recent_year.parquet)
